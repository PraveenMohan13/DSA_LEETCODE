# Backtracking Practice Flow


## LeetCode Practice Flow

________________________________________
🔹 General Backtracking Mindset

Always ask 3 questions before coding:
Choices? → What options can I pick at each step?
Constraints? → What options should I avoid?
Goal? → When do I stop and add to result?

Use recursion tree drawing (pen & paper) for n=3 → it will make the pattern crystal clear.
________________________________________
